# AI-Based Scheme Categorizing System

This system automatically categorizes government schemes using NLP and K-means clustering, then notifies relevant users.

## Features

- Natural Language Processing for scheme analysis
- K-means clustering for automatic categorization
- User-friendly web interface
- MySQL database for data storage
- RESTful API backend
